# Protótipo

## Objetivo
Esta pasta documenta o protótipo interativo criado para testes de usabilidade e demonstração das soluções propostas.

## Conteúdo

### O que você encontrará aqui:
- **Link do Protótipo**: Acesso ao protótipo no Figma
- **Instruções de Navegação**: Guia para interagir com o protótipo
- **Cenários de Teste**: Tarefas para validação com usuários
- **Resultados dos Testes**: Feedbacks e aprendizados
- **Iterações**: Versões e melhorias aplicadas

## Detalhes do Protótipo

### Ferramenta Utilizada
**Figma** - Plataforma colaborativa de design e prototipagem

### Tipo de Protótipo
- **Fidelidade**: Alta fidelidade (visual completo)
- **Interatividade**: Protótipo clicável com transições
- **Escopo**: Principais fluxos de usuário

### Fluxos Prototipados
1. **Descoberta e Assinatura do Meli+**
   - Landing page com benefícios
   - Seleção de plano
   - Cadastro de meio de pagamento
   - Confirmação da assinatura

2. **Gestão de Meios de Pagamento**
   - Visualização de métodos cadastrados
   - Adição de novo cartão
   - Edição e remoção de métodos
   - Definição de método padrão

3. **Dashboard do Assinante**
   - Visão geral dos benefícios
   - Histórico de uso
   - Configurações da assinatura

## Testes de Usabilidade

### Objetivos dos Testes
- Validar a clareza da proposta de valor
- Verificar a facilidade do processo de assinatura
- Identificar pontos de fricção
- Avaliar a compreensão dos benefícios

### Métricas Avaliadas
- Taxa de conclusão de tarefas
- Tempo para completar fluxos
- Número de erros
- Satisfação do usuário (SUS)
